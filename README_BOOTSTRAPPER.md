# Інструкція з використання бутстрапперів (Bootstrappers)

Для того, щоб ваші гравці не отримували помилку "Библиотеки JavaFX не найдены", ви можете розповсюдити ці скрипти. Вони автоматично завантажують потрібну версію Java (BellSoft Liberica з підтримкою JavaFX) та запускають лаунчер.

## Файли

1.  **launcher_windows.bat** — для користувачів Windows.
2.  **launcher_linux.sh** — для користувачів Linux.

## Як налаштувати

У обох файлах на початку є блок налаштувань:

```bash
PROJECT_NAME="NestWorld"
LAUNCHER_URL="https://launcher.nestworld.site/Launcher.jar"
```

Переконайтеся, що `LAUNCHER_URL` вказує на пряме посилання для завантаження вашого `Launcher.jar`.

## Як використовувати гравцям

### Windows:
Гравцю потрібно просто запустити файл `launcher_windows.bat`.
Скрипт сам створить папку в `%APPDATA%\.NestWorld`, завантажить туди Java та лаунчер, і запустить його.

### Linux:
Гравцю потрібно:
1. Завантажити `launcher_linux.sh`.
2. Надати йому права на виконання: `chmod +x launcher_linux.sh`.
3. Запустити його: `./launcher_linux.sh`.

## Переваги
- Гравцям не потрібно самостійно встановлювати Java.
- Використовується BellSoft Liberica Full, яка гарантовано містить JavaFX.
- Лаунчер завжди запускається з правильними параметрами.
